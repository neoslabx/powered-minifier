Reduce your page load by minifying your HTML source along with all the CSS and JS code present in your markup.

== Description ==

Powered Minifier allow you to reduce your page load and increase your performance by minifying your HTML source along with all the CSS and JS code present in your markup.

== Installation ==

1. Upload "powered-minifier" folder to the "/wp-content/plugins/" directory
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Click the "Powered Minifier" link located under your "Settings" section to configure the plugin options
