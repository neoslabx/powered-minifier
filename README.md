# Powered Minifier

![PHP Version](https://img.shields.io/badge/php-8.4%2B-blue)
![License](https://img.shields.io/badge/license-MIT-green)

Reduce your page load by minifying your HTML source along with all the CSS and JS code present in your markup.

* * *

## Description

Powered Minifier allows you to reduce your page load and increase your performance by minifying your HTML source along with all the CSS and JS code present in your markup.

* * *

## Installation

1. Upload the **`powered-minifier`** folder to the **`/wp-content/plugins/`** directory.
2. Activate the plugin through the **Plugins** menu in WordPress.
3. Click the **Powered Minifier** link located under your **Settings** section to configure the plugin options.
